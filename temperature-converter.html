How to Use

Basic Conversion

  1. Enter a temperature value in the "From" field
  2. Select the unit you're converting from (Celsius, Fahrenheit, Kelvin, or Rankine)
  3. Select the unit you want to convert to
  4. Click "Convert" or press Ctrl/Cmd + Enter

Keyboard Shortcuts

  • Ctrl/Cmd + Enter - Convert temperature
  • Ctrl/Cmd + R - Clear form
  • Ctrl/Cmd + S - Swap units
  • Ctrl/Cmd + C - Copy result to clipboard

Export Options

  • Click the "Export" button to see available options
  • Copy result to clipboard
  • Download as PNG or JPG image

Tips

  • Use the swap button to quickly switch between units
  • Check the reference points for common temperatures
  • Toggle dark/light theme using the moon/sun icon
  • Invalid temperatures (below absolute zero) will show an error

Conversion Formulas

Celsius to Fahrenheit

°F = (°C * 9/5) + 32

Celsius to Kelvin

K = °C + 273.15

Celsius to Rankine

°R = (°C + 273.15) * 9/5

Fahrenheit to Celsius

°C = (°F - 32) * 5/9

Fahrenheit to Kelvin

K = (°F + 459.67) * 5/9

Fahrenheit to Rankine

°R = °F + 459.67
